Font Manager

Source: GitHub

Change the font for all pages.

By selecting and enabling the font included in the extension, you can use that font on all pages.

Using familiar fonts will make surfing the internet even more comfortable.

We are currently looking for fonts to be used in the extension. Suggestions are accepted on Github or Discord. If you don't have a font you like, please contact us! We'll respond!

Any other font requests? Support on GitHub or Discord!
